BARRE D’OUTILS

 Mettre à jour la liste des notifications depuis Geocaching.com
 Créer une nouvelle notification de publication de cache (voir plus bas)
 Opérations sur une sélection de notification :

·         Sélectionner tout : sélectionne toutes les notifications

·         Désélectionner tout : désélectionne toutes les notifications

·         Inverser l’état d’activation : inverse l’état des notifications (active / en pause)

·         Changer le centre : permet de changer facilement les coordonnées du centre des notifications, soit avec la position courante, soit en choisissant un point sur la carte

·         Afficher sur la carte : affichage des notifications sur la carte (voir plus bas)

·         Supprimer : supprime les notifications

Touchez simplement une notification pour la sélectionner / désélectionner. Les notifications sélectionnées sont surlignées :
               
 Configuration de votre identifiant et mot de passe Geocaching.com. Cette opération n’est à effectuer normalement qu’au lancement de l’application. Le mot de passe est stocké crypté dans votre téléphone.
 Affiche les informations « A propos » de EzFTF ainsi que ses licences associées
 Quitter l’application

CHOIX DE COORDONNEES SUR LA CARTE

Un marqueur est positionné par défaut aux coordonnées renseignées dans le ‘Coordonnées’ (ou sur la ville de Marseille si les coordonnées sont non valides).

Pour le déplacer et modifier ses coordonnées :

·         effectuer un appui long sur celui-ci,

·         lorsqu’il se positionne au-dessus de votre doigt, vous pouvez le déplacer (maintenir l’appui long)

·         pour figer sa position, relâcher l’appui

AFFICHAGE DES NOTIFICATIONS SUR LA CARTE

Par défaut, l’ensemble des notifications sélectionnées sont visible à l’écran.

Il est possible de déplacer la carte, zoomer, etc.

Un marqueur indique le centre d’une (ou plusieurs) notifications, le disque coloré l’entourant indique le rayon associé.

Un clic sur le marqueur permet d’en afficher les détails (nom de la notification, rayon et type de cache concerné). Lorsque plusieurs notifications partagent le même centre et le même rayon, elles sont ‘colocalisées’ (voir ci-dessous).

Couleurs employés pour les notifications :

Traditional cache / Letterbox Hybrid

Mystery / Wherigo

 Earthcache / Virtual / Webcam

Event / Mega / Giga / type non supporté

Multi-cache
